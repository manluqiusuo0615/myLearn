命令格式与目录处理命令ls
命令格式:  命令 -[-选项] [参数]
    选项:调整命令功能的
    参数:命令操作的对象(用yu'i户,进程,文件等等)
    用[]括起来,说明不
    例: ls -la /etc
说明:1）个别命令使用不遵循此格式
    2）当有多个选项时,可以写在一起
        ls -lh /
        人性化显示/下面的文件详细信息
    3）简化选项与完整选项
        -a 等于 --all
        -h 等于 --human(人类,人性化)

1).目录处理命令:ls
    命令名称:ls
    命令英文原意:list
    命令所在路径:/bin/ls
    执行权限:所有用户
    功能描述:显示目录文件
    语法:ls 选项[-ald] [文件或目录]
            -a 显示所有文件,包括隐藏文件
            -l 详细信息显示
            -d 查看目录属性(direct),只看目录,而不看其中子目录
            -i inode(i节点号)
在Linux中,以"."开头的文件都为隐藏文件
隐藏文件的初衷,是告诉用户,该隐藏文件为系统文件,不要随意操作
a ===  all
l ===  long（长格式显示）

例:
ls -l
total 4
-rw-------.   1   root   root   1666   Oct 21 20:24   anaconda-ks.cfg
描述了文件类型和权限(root有rw权限,g因为对应---,所以没有任何权限,o同理)   文件的引用计数   所有者   所属组  文件大小(默认单位为字节)   修改时间   文件名称
-rw-r--r--
-文件类型(-二进制文件 d目录 l软链接文件)
rw- r-- r--
u   g   o
u所有者  g所属组  o其他人
r读  w写  x执行

Linux中,每个文件把用户分成了三类:
    1.所有者  u  user （只能有一个）一般来说创建了这个文件,就是这个文件的所有者
        所有者的身份是可以变化的
    2.所属组  g  group （）
    3.其他人  o  other
用户与文件之间的关系
2).目录处理命令:mkdir
    命令名称:mkdir
    命令英文原意:make directories
    命令所在路径:/bin/mkdir
    执行权限:所有用户
    功能描述:创建新目录
    语法:ls mkdir -p [目录名]
            -p 递归创建
3).目录处理命令:pwd
    命令名称:pwd
    命令英文原意:print working directory
    命令所在路径:/bin/pwd
    执行权限:所有用户
    功能描述:显示当前目录
    语法:pwd
4).目录处理命令:cd
    命令名称:cd
    命令英文原意:change directory
    命令所在路径:shell内置命令
    执行权限:所有用户
    功能描述:切换目录
    语法:cd [目录]
.表示当前目录  ..表示当前目录的上一级目录
5).目录处理命令:rmdir
    命令名称:rmdir
    命令英文原意:remove empty directory
    命令所在路径:/bin/rmdir
    执行权限:所有用户
    功能描述:删除空目录
    语法:rmdir [目录]
6).目录处理命令:cp
    命令名称:cp
    命令英文原意:copy
    命令所在路径:/bin/cp
    执行权限:所有用户
    功能描述:复制目录或文件
    语法:cp -rp [原文件或目录] [目标目录]
            -r 复制目录
            -p 保留文件属性
